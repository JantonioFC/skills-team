# Ejemplos de Uso para el Creador de Habilidades

Aquí tienes 20 ejemplos de cómo puedes pedirle a tu agente que utilice esta habilidad para crear nuevas herramientas.

## Desarrollo Web y Frontend
1.  "Crea una habilidad llamada 'Experto en React' que se centre en buenas prácticas de hooks y componentes funcionales."
2.  "Genera una skill para 'Tailwind CSS' que siempre sugiera el orden correcto de clases y configuraciones de tema optimizadas."
3.  "Necesito una habilidad de 'Accesibilidad Web (A11y)' que revise mi HTML en busca de atributos ARIA faltantes y contraste de colores."
4.  "Haz una skill de 'Optimización de Next.js' para guiarme en el uso de Server Components y optimización de imágenes."

## Backend y DevOps
5.  "Crea una habilidad para 'Dockerización' que me ayude a escribir Dockerfiles multi-stage eficientes para Node.js y Python."
6.  "Genera una skill de 'Seguridad API' que audite endpoints buscando vulnerabilidades comunes como inyección SQL o falta de autenticación."
7.  "Quiero una habilidad de 'Gestión de Kubernetes' para generar archivos YAML de despliegue y servicios rápidamente."
8.  "Crea una skill llamada 'Arquitecto de Microservicios' que me ayude a diseñar patrones de comunicación entre servicios."

## Data Science e IA
9.  "Haz una skill de 'Limpieza de Datos con Pandas' que tenga scripts listos para manejar valores nulos y duplicados."
10. "Crea una habilidad de 'Ingeniería de Prompts' que me ayude a refinar mis instrucciones para modelos de lenguaje."
11. "Necesito una skill de 'Visualización con Matplotlib' que sugiera los mejores gráficos según el tipo de datos."
12. "Genera una habilidad para 'Validación de Modelos ML' que verifique métricas como precisión, recall y F1-score."

## Automatización y Scripts
13. "Crea una skill de 'Automatización con Python' para generar scripts que renombren archivos masivamente y organicen carpetas."
14. "Haz una habilidad de 'Git Workflow' que me recuerde seguir Conventional Commits y git flow."
15. "Genera una skill para 'Web Scraping Ético' que configure Beautiful Soup o Selenium respetando el robots.txt."

## Productividad y Organización
16. "Crea una habilidad llamada 'Resumidor de Reuniones' que tome notas crudas y las convierta en una minuta estructurada."
17. "Necesito una skill de 'Generador de README' que cree documentación de proyectos atractiva y completa automáticamente."
18. "Haz una skill de 'Planificación de Sprints' que me ayude a dividir tareas grandes en user stories pequeñas."
19. "Crea una habilidad de 'Revisor de Ortografía y Estilo' para corregir documentación técnica en español."

## Validación y Mantenimiento
20. "Por favor, valida todas las skills en mi carpeta actual usando el script de validación del Creador de Habilidades."
